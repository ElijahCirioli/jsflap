# jsFLAP

A javascript tool for creating finite state automata

Hosted at https://elijahcirioli.com/jsflap/

## Features

-   DFAs and NFAs
-   Pushdown automata
-   Turing machines
-   NFA -> DFA conversion
-   NFA/DFA equivalence testing
-   Layout algorithms
-   Step-by-step parsing
-   Regular expression -> NFA conversion
-   Automaton cross product
