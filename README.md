# jsFLAP

A javascript tool for creating finite state automata

Hosted at https://elijahcirioli.com/jsflap/

## Already implemented

-   DFAs and NFAs
-   Pushdown automata
-   NFA -> DFA conversion
-   NFA/DFA equivalence testing
-   Layout algorithms

## Not yet implemented

-   Product automata
-   Turing machines
-   Step-by-step parsing
